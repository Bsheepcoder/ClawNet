# Windows 快速开始指南

## 🚀 1分钟安装

### 步骤1：安装

打开 PowerShell 或 CMD：

```bash
npm install -g @husile/clawnet-lite
```

### 步骤2：验证

```bash
clawnet --version
```

### 步骤3：使用

```bash
# 启动交互式 CLI
clawnet-interactive

# 或使用命令行
clawnet --help
clawnet health
```

---

## ✅ 完成！

现在你可以在任何地方使用 `clawnet` 命令。

## ❓ 遇到问题？

如果 `clawnet` 命令不可用：

```bash
# 检查安装位置
npm config get prefix

# 添加到 PATH
# 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
# 在 Path 中添加：C:\Users\你的用户名\AppData\Roaming\npm
```
